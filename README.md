# Hi, I'm Vadim 👋

I build test frameworks and tooling that catch bugs before users do. Started in
manual QA, went deep into automation — **mobile** (Appium, XCUITest,

## 🚀 Featured projects

### 🤖 [Mobiscout](https://github.com/VadimToptunov/Mobiscout)
AI-powered mobile test-automation framework with **self-healing** locators —
Android & iOS, BDD, CI/CD ready.

### 📱 [KassiOS](https://github.com/VadimToptunov/KassiOS)
A tiny, batteries-included **Kaspresso-style DSL over XCUITest** for readable,
maintainable iOS UI tests.

### 🏦 ChaosBank — an end-to-end testing playground
A *deliberately-buggy* neobank, built to be tested every which way:
- [ChaosBank-iOS](https://github.com/VadimToptunov/ChaosBank-iOS) · [ChaosBank-Android](https://github.com/VadimToptunov/ChaosBank-Android) — the apps
- [ChaosBank-KassiOS-Tests](https://github.com/VadimToptunov/ChaosBank-KassiOS-Tests) — hand-written XCUITest suite
- [ChaosBank-Mobiscout-Tests](https://github.com/VadimToptunov/ChaosBank-Mobiscout-Tests) — tests auto-generated by Mobiscout

### 🗺️ [Canada Real-Estate Analytics](https://github.com/VadimToptunov/canada_real_estate_analytics)
Data pipeline + interactive map of one-bedroom rent prices across Canadian cities.
